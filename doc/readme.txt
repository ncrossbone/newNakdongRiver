낙동강 물환경 페이지를 작성시는 기존의 개발 방법과 다릅니다.아래와 같은 순서대로 코딩을 하셔야 합니다. 
1. 이동할 페이지는 현재 contextRoot 가 /nakdong 으로 되어 있기 때문에 이동할려고 하는 링크정보에 포함되어야 함
--------------------------------------------------------------------------------------------------------------------------------------------------------
2. 이동할 경우 링크정보는 반드시 *.do 확장자를 사용하셔야 합니다.  그리고 해당 링크정보는 web.xml 에 바로 등록을 하셔야 합니다. 
등록하는 방법은 서블릿 링크 등록방법과 동일한 방법으로 하셔야 합니다.
예제) 
 <servlet>
    <servlet-name>envBrdWrite2</servlet-name>
    <jsp-file>/jsp/board/env_brd_write2.jsp</jsp-file> <!-- 실제 보여지는 Jsp 경로 -->
  </servlet>
  <servlet-mapping>
    <servlet-name>envBrdWrite2</servlet-name>
    <url-pattern>/envBrdWrite2.do</url-pattern> <!-- 페이지 이동시 링크정보 -->
  </servlet-mapping>

--------------------------------------------------------------------------------------------------------------------------------------------------------
즉 페이지 이동을 할 경우 링크정보는 *.do 로 하고 web.xml 에 등록후에 페이지 이동이 되는것을 확인하고 해당 데이타는
--------------------------------------------------------------------------------------------------------------------------------------------------------
Ajax 를 이용해서 해당 정보를 가져와야 합니다. 이때 링크 확장자는 .json 을 사용하셔야 해당 컨텐츠 정보를 가져올수 있습니다.
